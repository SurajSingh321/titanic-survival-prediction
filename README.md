# Titanic Survival Prediction

A simple ML project to predict whether a passenger survived the Titanic or not.

I built this using Decision Tree on the Titanic dataset from Kaggle.

## What it does

Takes passenger details like age, gender, class, fare etc. and predicts whether they survived or not.

## Tech used

- Python
- Scikit-learn (Decision Tree)
- Pandas, NumPy

## How to run

pip install pandas scikit-learn numpy
python titanic_survival.py

Make sure Titanic-Dataset.csv is in the same folder.

## Dataset

Titanic Dataset from Kaggle — 891 passengers.

## Accuracy

Got around 80% accuracy on the test set.
